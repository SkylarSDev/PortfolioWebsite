Dandy's World VRC

Dandy's World VRC is a port of Dandy's World by BlushCrunch Studio made entirely in VRChat using Udon and Unity. This project is a passion project and collaboration between myself and several others with the goal of bringing a game we all love to a platform we all spend a lot of time in. On this project, I am the one doing the entire Unity side of things, including the programming. :3
